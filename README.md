### 算法

#### 时间复杂度

> 一个算法的时间复杂度反映了程序运行从开始到结束所需要的时间。把算法中基本操作重复执行的此处作为算法的时间复杂度。

- `O(1)`: 常数复杂度
- `O(log n)`: 对数复杂度
- `O(n)`: 线性时间复杂度
- `O(n^2)`: 平方复杂度
- `O(n^3)`: 立方复杂度
- `O(2^n)`: 指数复杂度
- `O(!n)`: 阶乘复杂度

#### 空间复杂度

> 空间复杂度是指运行完一个程序所需内存的大小。利用程序的空间复杂度，可以对程序的运行所需内存有预先估计。
